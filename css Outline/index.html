CSS OUTLINE PROPERTY

CSS Outline

The outline property is used to make another border it means outside of the border of any element. we can make two line at once of any element in html.

Outline-style

The Css outline style is used to make the outline in different styles the outline-style property has same values as border values.for example:

  • solid
  • dotted
  • dashed
  • double
  • groove
  • ridge
  • inset
  • ofset
  • none
  • hidden
