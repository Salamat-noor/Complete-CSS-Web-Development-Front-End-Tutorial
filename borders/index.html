BORDERS

A border is a basic layer which covers the all content of an html element it shows the boundries of an element and also it shows the width and height of that element.

BORDER STYLE

There are multiple border styles. We can make the border manually with all four sides (top,right,bottom,left.)

  • solid: Defines a solid line border


  • dotted: Defines a dotted border


  • dashed: Defines a dashed dashed border


  • double: Defines a border like solid but with dual lines like a path i said.


  • groove: Defines a 3D grooved border


  • ridge: Defines a 3D ridged border.


  • inset: Defines a 3D inset border.


  • outset: Defines a 3D outset border.


  • none: means no border to apply.


  • hidden: Defines a border but hidden to visible.
