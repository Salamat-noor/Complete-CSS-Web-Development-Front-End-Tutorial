CSS Advanced Text-Effects

CSS Text-Effect Properties

Text-ovrflow

The CSS text-overflow property have two values, clip & ellipsis It is used to make a text or a paragraph hide when it's parent element reaches max-width if text is too long to fit in it in html

word-wrap

The CSS word-wrap property is have one value break-word. It makes a word break to new line when it's length is too big to fit in its parent element's width it makes the long long words break to new lines

word-break

The CSS word-break property have two values, break-all & keep-all. It is used to make whole sentence justify in it's parent element's body it acts like word-wrap when we set it's value to break-all and It also makes whole alphabets fit in the given width of it's parent's element when value is set'ed to keep-all.

writing-mode

The CSS writing-mode have two or more values horizontal-tb,vertical-rl It is used to show the written word in different directions like top-to-bottom and right-to-left in order to vertically. for-example this has vertically writing-mode
