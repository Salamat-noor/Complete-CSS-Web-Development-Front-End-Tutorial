The CSS rotate() function is used to make an element rotate within clockwise and also anti-clockwise it makes an element like moving tyre when the value is setted.

CSS Transform scale() function

CSS Transform scaleX() function

The CSS scaleX() function is used to makes the width increase or decrease with the given default width It is used for only width increment or decrement

CSS Transform scaleY() function

The CSS scaleY() function is the same opposite of scaleX() function it is used to increase or decrease height of an element.
