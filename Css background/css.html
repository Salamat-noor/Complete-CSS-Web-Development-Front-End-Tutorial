Background Property

The Css background property is the short hand property of all background properties. It's have many properties but we can merge them and write them in our code in a ine line.Some of properties are given below.

background-color :
The background-color property is used when we have to use the color of our pages background or an elements background it makes the view more attractive.
background-image :
The background image property is used to add images to the web page or html Elements we can add single image or multiple images to that section In this body we have to make images in this property with commas after first url value The image is set by background-image property.
background-size :
The background-size property is used to set the size for the background-image There are multiple values we can set the size by x-axis and y-axis / width / height, also we can set it for one by one it means for all images in the background-image with seperated by commas for each.
background-position :
We can set the position for each image and by default it hides the multiple images backside of each image. Or we can drag the images with numeric values first one will for x-axis and second value will be for y-axis. Or it have some default values like :
  • left
  • center
  • right
  • top
  • bottom
background-attachment :
it is one of the background-properties it makes the image scroll or fixed with the body or browsers page.It have two values fixed , scroll scroll means the image will scrolls and finished its size with the rest of the page.
background-repeat :
It is one of the most useable property it makes the background-image like multiple screens it repeats the image in the screen size which in it locates. it have multiple values but we uses the most are repeat (default), no-repeat
